About Our System

Understanding public sentiment through advanced AI-powered analysis

System Overview

The Marcos Jr. Sentiment Analysis System is a comprehensive web-based platform designed to collect, analyze, and visualize public sentiment regarding Philippine President Ferdinand "Bongbong" Marcos Jr. Our system leverages advanced machine learning techniques to provide real-time insights into public opinion.

This platform serves as a valuable tool for understanding the pulse of public sentiment, enabling researchers, policymakers, and citizens to gain insights into how the current administration is perceived by the Filipino people.

How It Works

1
Data Collection

Users submit feedback and ratings through our secure web interface.

2
AI Analysis

Our trained machine learning model analyzes text for sentiment classification.

3
Data Storage

Results are securely stored in our database for analysis and reporting.

4
Visualization

Administrators can view comprehensive analytics and insights through our dashboard.

Technical Features

  • Machine Learning Integration: Trained sentiment analysis model.
  • Real-time Analysis: Instant sentiment classification of user feedback
  • 5-Star Rating System: Complementary rating mechanism for comprehensive feedback
  • Interactive Dashboard: Visual analytics with charts and word clouds
  • Secure Authentication: Admin-only access to sensitive analytics
